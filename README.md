# AISI.services

**AISI.services** is the public AI-powered digital-services gateway of **American Communications Online (ACO)**.

The operating chain is:

```text
Theresa J. Morris (TJ Morris) — human founder and final publication authority
        ↓
JET Services — practical operations and routing desk
        ↓
AISI.services — public AI, web, media, and digital-services gateway
```

JET Services is an operations layer under TJ Morris. It is not presented as an autonomous authority.
