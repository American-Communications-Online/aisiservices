# AISI.services Static Deployment Notes

This repository contains the public static website source for AISI.services.

## Source files

- `index.html` — public landing page
- `404.html` — fallback page
- `_headers` — baseline browser security headers
- `robots.txt` — crawler rule
- `sitemap.xml` — basic site map

## Publication boundary

Theresa J. Morris (TJ Morris) remains the human publication gate. JET Services is the practical operations and routing desk. AISI.services is the public delivery gateway.

## Hosting note

The production host must be configured to publish the repository root from the `main` branch. The exact hosting configuration should be maintained privately by the authorized site administrator.
