## Verification checklist

1. Open the public URL in a browser without signing in.
2. Confirm the visible display name and channel branding.
3. Record whether the account is active, legacy, redirect-only, or archive-only.
4. Confirm that the authorized administrator can sign in.
5. Store passwords, recovery methods, phone numbers, and backup codes only in a secure private location.
6. Publish only the verified public URL in `PUBLIC-CHANNEL-REGISTRY.md`.
