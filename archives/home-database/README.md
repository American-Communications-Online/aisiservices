# AISI.services Home Database Archive

This folder is the public-safe archive index for AISI.services.

It is designed to organize public records, framework pages, agent registries, research notes, media references, and reusable templates without mixing private material into the public repository.

## Public Archive Lanes

| Lane | Folder | Purpose |
| --- | --- | --- |
| A | `agents/` | Public-safe agent names, roles, and registry notes |
| B | `frameworks/` | Public symbolic-ontology and architecture records |
| C | `research/` | Public research notes and source-based summaries |
| D | `media/` | Public media references, shows, and publishing records |
| E | `templates/` | Reusable archive-record templates |

## Important Boundary

This repository is public. Do not place passwords, recovery codes, private messages, personal medical records, financial records, unpublished witness details, or restricted archive material in this folder.

Private and restricted records should be stored outside the public GitHub repository and referenced here only by a public-safe title when appropriate.
